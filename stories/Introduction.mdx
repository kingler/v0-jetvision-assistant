# Jetvision Component Library

Welcome to the **Jetvision AI Assistant** component library documentation. This library provides a comprehensive set of UI components built for the private aviation charter platform.

## Design System

Our design system is built on the following principles:

### Colors

- **Primary (Sky Blue)**: `#00a8e8` - Innovation, Speed, Trust, Professionalism
- **Accent (Sunset Orange)**: `#ff6b35` - Luxury, Premium
- **Aviation Blue**: `#0066cc` - Legacy brand color

### Typography

- **Sans-serif**: Arial, sans-serif (clean, professional)
- **Monospace**: "Courier New", monospace (data, code)

### Accessibility

All components are built with WCAG 2.1 AA compliance:
- Minimum touch targets of 44x44px
- Color contrast ratios of 4.5:1 for normal text
- Enhanced focus indicators for keyboard navigation
- Screen reader support with proper ARIA attributes

## Component Categories

### Foundation
- **Button** - Primary interactive element
- **Input** - Text input fields
- **Card** - Content containers
- **Badge** - Status indicators

### Chat Components
- **MessageBubble** - Chat message display
- **MessageList** - Scrollable message container

### Aviation-Specific
- **AviondeRFPCard** - RFP flow cards
- **TripBadge** - Trip status indicators

## Getting Started

```tsx
import { Button } from '@/components/ui/button';
import { Card, CardHeader, CardTitle, CardContent } from '@/components/ui/card';

function MyComponent() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>Welcome</CardTitle>
      </CardHeader>
      <CardContent>
        <Button>Get Started</Button>
      </CardContent>
    </Card>
  );
}
```

## Theming

Components support light and dark modes via the `dark` class on the root element. Use the theme switcher in the toolbar to preview components in different themes.
