## Typography

### Font Families

| Token | Value | Usage |
|-------|-------|-------|
| `font-sans` | Arial, sans-serif | Body text, UI elements |
| `font-mono` | "Courier New", monospace | Code, data displays |

### Font Sizes

| Token | Size | Usage |
|-------|------|-------|
| `text-xs` | 0.75rem (12px) | Labels, metadata |
| `text-sm` | 0.875rem (14px) | Captions, small text |
| `text-base` | 1rem (16px) | Body text |
| `text-lg` | 1.125rem (18px) | Large body text |
| `text-xl` | 1.25rem (20px) | Subheadings |
| `text-2xl` | 1.5rem (24px) | Section headers |
| `text-3xl` | 1.875rem (30px) | Page titles |
| `text-4xl` | 2.25rem (36px) | Hero text |

## Border Radius

| Token | Value | Usage |
|-------|-------|-------|
| `radius-sm` | 4px | Small elements, badges |
| `radius-md` | 6px | Inputs, buttons |
| `radius-lg` | 8px | Cards, containers |
| `radius-xl` | 12px | Dialogs, panels |
| `radius-full` | 9999px | Pills, avatars |

## Shadows

| Token | Usage |
|-------|-------|
| `shadow-xs` | Subtle elevation |
| `shadow-sm` | Cards, buttons |
| `shadow-md` | Dropdown menus |
| `shadow-lg` | Dialogs, popovers |
| `shadow-xl` | Modal overlays |

## Breakpoints

| Token | Value | Usage |
|-------|-------|-------|
| `sm` | 640px | Mobile landscape |
| `md` | 768px | Tablet |
| `lg` | 1024px | Desktop |
| `xl` | 1280px | Large desktop |
| `2xl` | 1536px | Wide screens |

## Z-Index Scale

| Token | Value | Usage |
|-------|-------|-------|
| `z-base` | 0 | Default |
| `z-dropdown` | 1000 | Dropdown menus |
| `z-sticky` | 1100 | Sticky headers |
| `z-modal` | 1400 | Modal dialogs |
| `z-popover` | 1500 | Popovers |
| `z-tooltip` | 1600 | Tooltips |

## Accessibility

### Focus Ring

All interactive elements have visible focus indicators:
- Width: 3px
- Offset: 2px
- Color: Sky Blue (ring color)

### Touch Targets

- Minimum: 24×24px (WCAG 2.1 AA)
- Recommended: 44×44px (WCAG 2.1 AAA)

### Color Contrast

- Normal text: 4.5:1 minimum (WCAG AA)
- Large text: 3:1 minimum (WCAG AA)
