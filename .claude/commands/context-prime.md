# Context Prime

Invoke the `context-prime` skill with focus mode: `$ARGUMENTS`

If no argument is provided, use `full` mode.

**Usage examples:**
- `/context-prime` — Full project context
- `/context-prime ui` — UI/frontend focus
- `/context-prime backend` — Backend/agents focus
- `/context-prime api` — API routes and MCP servers
- `/context-prime database` — Supabase migrations and schema
- `/context-prime "webhook flow"` — Custom keyword search
